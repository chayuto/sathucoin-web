# Thai Language Glossary for AI Coders

Reference for writing natural Thai UX copy in SaThuCoin. Based on real-world usage from Bitkub, PunBoon (ปันบุญ), Krungthai เติมบุญ, SCB Easy, and Thai temple donation sites.

## Rules

1. **Keep English for**: wallet, MetaMask, Base, blockchain brand names, token symbols (SATHU), BaseScan, GitHub, QR code, Layer 2
2. **Translate to Thai for**: actions, descriptions, labels, headings, FAQs, instructions
3. **Tone**: Polite but not overly formal. Use ครับ/ค่ะ sparingly (only in conversational FAQ answers if at all). Avoid robotic/literal translations.
4. **Buddhist context**: This is a merit-making (ทำบุญ) app. Use ทำบุญ and สาธุ naturally. Thai donors understand these concepts deeply -- don't over-explain them.

## Donation/Merit-Making Terms

Source: PunBoon (ปันบุญ), Krungthai เติมบุญ, Thai temple sites

| English | Thai (use this) | Notes |
|---------|-----------------|-------|
| Donate / Donation | บริจาค / การบริจาค | Most common, neutral. Used by all platforms. |
| Make merit | ทำบุญ | THE key Thai term. Deeply cultural. Implies spiritual goodness. Use for the "deed" concept. |
| Merit / Good deed | บุญ | "บุญ" alone means merit/virtue |

| Charitable giving | การให้ / การทำบุญทำทาน | "การทำบุญทำทาน" is the full Buddhist phrase |
| Recognition | การยกย่อง / เป็นเกียรติ | |
| Proof of donation | หลักฐานการบริจาค | |
| Record | บันทึก | |
| Permanent record | บันทึกถาวร | |

## Organization/Institution Terms

Source: Thai Revenue Department, PunBoon, temple sites

| English | Thai (use this) | Context |
|---------|-----------------|---------|
| Institution | สถาบัน | Generic, slightly formal |
| Organization | องค์กร | More common in charitable context |
| Foundation | มูลนิธิ | Legal term for registered foundations |
| Charitable org | องค์กรสาธารณกุศล | Official Revenue Dept term |
| Temple | วัด | Most common donation recipient in Thailand |
| Hospital | โรงพยาบาล | Common donation recipient |
| School | โรงเรียน / สถานศึกษา | |
| Participating org | องค์กรที่เข้าร่วม / หน่วยงานที่เข้าร่วม | |

**Recommendation for SaThuCoin**: Use "องค์กร" as the primary term instead of "สถาบัน". It's warmer, broader, and matches how PunBoon and Krungthai เติมบุญ address recipient organizations. "สถาบัน" sounds academic.

## FAQ Writing Style

Thai FAQ questions should feel conversational, like a friend asking. Avoid stiff/bureaucratic phrasing.

| Stiff (avoid) | Natural (use this) |
|---------------|-------------------|
| มีค่าใช้จ่ายอะไรไหม? | เสียค่าใช้จ่ายไหม? / ต้องเสียเงินไหม? |
| ฉันทำอะไรกับ SATHU ได้บ้าง? | SATHU ใช้ทำอะไรได้บ้าง? |
| ฉันไม่เห็นโทเคนของฉัน? | ทำไมไม่เห็นโทเคน? / ไม่เห็นโทเคนในกระเป๋า? |
| อยู่บนเครือข่ายไหน? | SATHU อยู่บนเครือข่ายอะไร? |
| เราต้องจัดการคริปโตเคอร์เรนซีไหม? | องค์กรต้องยุ่งกับคริปโตไหม? |
| ควรแนะนำ wallet อะไรให้ผู้บริจาค? | แนะนำ wallet อะไรให้ผู้บริจาคดี? |

**Tone rule**: Use "ไหม" (casual question particle) not "หรือไม่" (formal). Drop "ฉัน" where possible -- Thai often omits the subject.

## Common Pitfalls

1. **Don't translate "wallet" to "กระเป๋าสตางค์"** -- Thai crypto users say "wallet" in English or "กระเป๋า"
2. **Don't use "คริปโตเคอร์เรนซี"** for every mention -- just say "คริปโต" or avoid it entirely when context is clear
3. **Don't say "ที่อยู่สัญญา"** for contract address -- say "ที่อยู่ Smart Contract" or "ที่อยู่สัญญาอัจฉริยะ" or just "Contract Address"
4. **Don't say "สร้างแล้ว" for minted** -- "สร้างแล้ว" is OK but "ที่ออกแล้ว" (issued) also works for token context
5. **Don't translate "smart contract" literally** -- Thai users know the English term
6. **"สถาบัน" sounds cold** -- use "องค์กร" (organization) or be specific: "วัด" (temple), "มูลนิธิ" (foundation)
7. **"การกระจายอุปทาน" is too economics-textbook** -- say "สัดส่วนโทเคน" (token proportion) instead
8. **"ขีดจำกัด" is fine** but "ลิมิต" (transliteration of "limit") is also common in casual Thai
